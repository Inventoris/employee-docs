# Сущности

В сервисе сотрудников Employee есть несколько сущностей. Все они привязаны к пользователю и остаются общими для его организаций. Между пользователями сущности не передаются.

Вот на какие виды они делятся:

## `Person`

Сотрудник.

| Атрибут | Значение |
|---------|----------|
| Таблица | `employee.persons` |
| Ключ | `Guid`: идентификатор сотрудника |
| Поля | `FullName`: полное имя (ФИО). <br> `InnFl`: ИНН физического лица. <br> `Gender`: пол сотрудника. <br> `Citizenship`: гражданство. <br> `BirthDate`: дата рождения. <br> `BirthPlace`: место рождения. <br> `Phone`: номер телефона. <br> `Email`: адрес электронной почты. <br> `RegistrationAddress`: адрес регистрации. <br> `IdentityCard`: документ, удостоверяющий личность. <br> `ForeignAddress`: адрес проживания за пределами страны. |

---

## `Position`

Должность сотрудника в организации. Один сотрудник может иметь только одну должность внутри одной организации. 

| Атрибут | Значение |
|---------|----------|
| Таблица | `employee.positions` |
| Ключи | `Person`: guid сотрудника <br> `Organization`: guid организации |
| Поля | `Title`: название должности. |

---

## `Role`

Роль сотрудника в организации (как в своей, так и в другой).

| Атрибут | Значение |
|---------|----------|
| Таблица | `employee.roles` |
| Ключи | `Organization`: guid организации <br> `Type`: тип роли (`RoleType`: `Head`, `ChiefAccountant`, `Sender`, `Ip`) |
| Поля | `Person`: guid сотрудника с ролью <br> `PositionOrganization`: guid организации, где работает сотрудник с данной ролью |
