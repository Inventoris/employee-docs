# Employee Docs

Документация к внутреннему сервису Employee.

## Что это за сервис

Employee — это WebAPI-сервис для работы с данными сотрудников. Он позволяет хранить, читать и редактировать их информацию.

С его помощью удалось решить проблемы на стыке разных сервисов. Например, раньше данные сотрудников хранились отдельно в каждой организации, из-за чего возникали дубли, сложности с обновлением и несогласованность.

## Ключевые возможности

Ниже — три главных возможности сервиса:

- единое хранение данных сотрудников;
- гибкая связь между сотрудниками и организациями;
- удобный доступ к данным через клиент и REST API.
