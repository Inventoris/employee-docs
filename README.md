# Employee Docs

Документация к внутреннему сервису Employee.

## Что это за сервис

Employee — это WebAPI-сервис для работы с данными сотрудников. Он позволяет хранить, читать и редактировать их информацию.

С его помощью удалось решить две главные проблемы:

1. Раньше данные сотрудников хранились отдельно в каждой организации, из-за чего возникали дубли и сложности с обновлением.
2. Сотрудникам приходилось несколько раз вводить реквизиты одной организации, причём любые изменения не синхронизировались.

Благодаря Employee информация о сотрудниках теперь едина для всех организаций одного пользователя, а на стыке нескольких сервисов нет рассинхронизации.

## Ключевые возможности

Ниже — три главных возможности сервиса:

- единое хранение данных сотрудников;
- гибкая связь между сотрудниками и организациями;
- удобный доступ к данным через клиент и REST API.
